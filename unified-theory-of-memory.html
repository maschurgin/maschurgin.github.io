A Unified Theory of Visual Memory

Schurgin, Wixted, & Brady (2018). Psychophysical Scaling Reveals a Unified Theory of Visual Memory Strength. bioRxiv, 325472
Summary
Quantitative models of working memory are generally based on the idea that people have a fixed capacity and do not represent all items, and thus performance arises from “mixtures” of remembered (i.e. ‘precision’) and not-remembered items (i.e. ‘capacity’). These models have been highly influential and utilized in hundreds of papers to draw important psychological conclusions. The impact of these models goes well beyond visual memory, as strong claims about the nature of attention and long-term memory, about deficits in clinical populations, and even about the underlying nature of consciousness have been made based on the supposedly discrete failures of memory proposed by these models. Critically, these models rely on a foundational assumption that the psychological representation of stimuli is based on physical stimulus space.

​In recent work, I have shown that how stimuli are actually perceived (and thus represented psychologically) relies on a transformed representation that is non-linearly related to physical stimulus space. This result calls into question the fundamental basis of nearly all extant models. Moreover, once this perceptual relationship is accounted for, I discovered that aspects of memory that previously required fundamentally different models – across different stimuli, tasks, and even the distinction between working and long-term memory – can be explained with a unitary signal detection framework (Target Confusability Competition Model or TCC). These results lead to a substantial reinterpretation of the relationship between perception, working memory and long-term memory, and reinforce the importance of considering how memory is shaped by perception.
Figure 1. (A) A widely used method in working memory is to select a color circle from a slice of color space, show working memory items drawn from this circle, and then at test probe the contents of a memory item by presenting the entire continuous circle to participants to make a response. Similar response wheels are used for other features, like orientation and face identity. (B) A histogram of results generally observed for such tasks, traditionally plotted as a function of physical distance, with the unit of measure being degrees of error along the response wheel. (C) In a triad psychological scaling task, participants had to say which of two colors in the bottom row was more similar to the top (target) color. Despite the difference between the two choice colors always being 30° on the color wheel, sensitivity (d′) dramatically decreased as the choices became more distant from the target. Error bars represent within-subject SEM. (D) We can use the data from the triad task, and from another psychological scaling task, the quad task, to infer the normalized psychological distance of colors at different physical distances along the color wheel (black o’s; black x’s). This function is decidedly non-linear, and thus these data contradict the linear scaling assumption of existing working memory models. Rather than using the quad or triad task and fitting a model, we can also ask participants to rate the similarity of two colors on a Likert scale (1-7) and normalize their ratings to range from 0 to 1, with 0 being most similar and 1 being least similar. This similarity rating task provides a smooth measure of psychological distance that is in close agreement with the results of the triad and quad task.
Figure 2. (A) Signal detection model applied to memory in a hypothetical 10-alternative forced-choice test. In standard 2-alternative long-term recognition memory experiments, “new” items sometimes feel more familiar or less familiar. The memory-match signals generated by new items are therefore modeled as a normal distribution with a mean of 0 and standard deviation of 1. By contrast, “old” items (encoded previously) elicit higher memory-match signals, modeled (in the simplest case) as a normal distribution with a mean of d′ and standard deviation of 1. Performance is best in conditions where d′ is high. In the case of a 10 alternative forced-choice, when asked to choose which of these alternatives corresponds to the old item, the simplest strategy is to sample a memory-match strength from each distribution and then choose the alternative that generates the highest memory signal. To specify this model, we thus only need to specify the memory-match strength of every lure. We place the mean of each lure’s memory-match distribution at its psychological distance to the target. In other words, if the target is purple, people will choose a slightly different purple lure much more often than an entirely unrelated lure (such as green in this example). The slightly different purple lure generates a much stronger memory-match signal than the green lure because it has a much shorter psychological distance to the target. Examples of d′=3 and d′=1 illustrate the idea that as memory signal strength for the target decreases, more of the lure distributions cluster near the target. The Target Confusability Competition Model (TCC) is simply an extension of the 10-AFC logic provided here, but adapted to every possible color probe (i.e. 360-AFC). If we assume that only a subset of colors are functionally probed at test (e.g., 60-AFC), the d′ estimates would change, but the fit of the model would not be appreciably affected. (B) TCC fits to data at set size 1, 3, and 6. With only a single parameter (d′), the TCC accurately captures the response distribution typically attributed to multiple parameters / psychological states by existing frameworks and models of working memory. On the left, we plot the probability of a response relative to the absolute value of physical error. A more appropriate way to plot these data is relative not to the physical distance, where it appears to be a normal distribution on top of a long tail, but to plot them relative to psychological distance (as shown on the right; which depicts the same bars plot -- that is, how often each response option was chosen -- but now with equal steps in psychological distance along the x-axis). In this space, there is neither a discontinuity nor a long-tail, indicative of why the TCC is able to fit this data straightforwardly. Note that plotting the data this way results in a clustering of the response distribution bars in the 90° to 180° physical distance range because they all fall at almost the same psychological distance from the target.
